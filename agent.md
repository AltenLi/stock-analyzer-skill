# Agent 设计逻辑记录

## 项目概述

stock-analyzer skill：全球股票综合分析工具，支持A股、港股、美股，从基本面、新闻面、资金面、技术面四维分析，生成投资建议和HTML可视化报告。

## 核心设计决策

### 2026-03-05: 新增并购重组（M&A）分析模块

**需求背景**：对于存在并购/收购/重组的公司，需要特殊优化判断逻辑，计算收购完成后的预期估值和股价涨幅空间。

**设计方案**：

1. **触发机制**：在 Step 4（新闻面分析）中检测并购相关关键词（收购、并购、重组、重大资产购买等），条件触发 Step 6.5 并购分析模块。

2. **数据获取**：通过 `web_search` 获取标的公司的营收、净利润、毛利率、收购比例、对价、支付方式、业绩承诺等关键数据。

3. **估值计算模型**：
   - **并表法**：收购方财务数据 + 标的财务数据 × 收购比例 = 并表后预期数据
   - **三场景估值**：乐观（行业PE 75分位）、中性（50分位）、保守（25分位）
   - **业绩承诺法**：基于承诺利润计算各年预期估值
   - **商誉风险评估**：商誉/净资产比率判断减值风险

4. **股价涨幅计算**：
   - 预期股价 = 预期市值 / 并购后总股本
   - 涨幅空间 = (预期股价 - 当前股价) / 当前股价 × 100%
   - 需同时考虑现金支付（不增发）vs 股份支付（增发摊薄）
   - **公告至今涨幅** = (当前股价 - 公告日收盘价) / 公告日收盘价 × 100%（直观反映市场对并购的实际定价反应）

5. **评分调整规则**：
   - 并购分析结果会调整基本面和新闻面评分
   - 营收/利润增厚 → 基本面加分
   - 商誉过高/标的亏损 → 基本面减分
   - 并购审批进展顺利 → 新闻面加分

6. **输出要求**：在报告中新增独立的"并购重组分析"板块（模板中为 Section 五(附)），包含并购概况、标的画像、三场景估值测算、股价涨幅空间、协同效应与风险评估。

**修改的文件**：
- `skill/SKILL.md` - Step 6.5 新增并购分析流程、数据获取、估值计算、协同评估、评分调整规则
- `skill/assets/report_template.md` - 新增 Section 五(附) 并购重组分析模板
- `skill/references/eastmoney_guide.md` - 新增并购估值计算方法、搜索策略、核心指标速查

## 评分体系

| 维度 | 权重 | 说明 |
|------|------|------|
| 基本面 | 35% | PE/PB/ROE/成长性/财务健康 |
| 新闻面 | 20% | 公告/研报/政策/舆情 |
| 资金面 | 35% | 主力净比(核心)、北向、融资 |
| 技术面 | 10% | 趋势/支撑压力/量价 |

> 并购分析不单独占权重，而是通过调整基本面和新闻面评分来体现影响。
